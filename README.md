# Music Track API - Java Spring Boot

Este es el backend de una aplicación de gestión de música, construido con **Java** y **Spring Boot**.

## Tecnologías utilizadas

- **Java 21**
- **React**
- **Spring Boot**
- **Spring Data** (Hibernate)
- **MySQL**
- **Maven**

## Endpoints Principales

- `GET /api/tracks` - Listar todas las canciones.
- `POST /api/tracks` - Añadir una nueva canción.
- `PUT /api/tracks/{id}` - Modificar una canción existente.
- `DELETE /api/tracks/{id}` - Eliminar una canción.
